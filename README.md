## 🚀 Featured Project

<details open>
<summary><b>🎯 Sonar Rock vs. Mine Detection</b></summary>
<br/>

Binary classifier distinguishing sonar signals reflected from rocks vs. metal mines.

| Stack | Scale | Performance | Impact |
|---|---|---|---|
| Python, Scikit-learn, Pandas, NumPy | 208 samples, 60 frequency-based features | 76.2% test accuracy (83.4% train) | Real-time single-instance prediction pipeline |

Trained a Logistic Regression model on the UCI Sonar dataset using a stratified 90/10 train-test split to preserve class balance, then built a single-instance inference pipeline for classifying new sonar readings in real time.

[View Repository](https://github.com/shashwat-sahuu)

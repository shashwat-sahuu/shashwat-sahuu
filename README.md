## 🚀 Featured Projects

<details>
<summary><b>🧬 GenomePipe Pro v4.0 — Clinical Genomics Platform</b></summary>
<br/>

Unified clinical genomics platform integrating 4 live NCBI databases (ClinVar, PubMed, GenBank, dbSNP) and Ensembl VEP into a single interface, replacing a 4-tool manual workflow.

| Stack | Scale | Performance | Security | Impact | Repository |
|---|---|---|---|---|---|
| JavaScript, FastAPI, IGV.js, Ensembl VEP, Ollama | 34 API routes, offline LLM assistant | Real-time variant lookup vs. 4 separate tools | Local/offline AI inference option | Won ICIBH 2026 | [View Repo](https://github.com/shashwat-sahuu) |

Implements a full ACMG/AMP 28-criteria variant classifier alongside a dual-mode AI clinical assistant that can run entirely offline via local Ollama LLMs, or via the Claude API — built for environments where clinical data cannot leave the local machine.

</details>

<details>
<summary><b>📊 CostPilot — AI Observability & Cost Management Platform</b></summary>
<br/>

A multi-tenant platform for observing and controlling AI/LLM spend, built with a native MCP server component so it plugs directly into AI-agent workflows.

| Stack | Scale | Performance | Security | Impact | Repository |
|---|---|---|---|---|---|
| FastAPI (Python), PostgreSQL, Redis, ClickHouse, MCP | Multi-tenant architecture | ClickHouse-backed analytics for high-volume event ingestion | Tenant-isolated data model | Flagship portfolio project | [View Repo](https://github.com/shashwat-sahuu) |

Designed as a 6–8 month build to demonstrate systems-level engineering: tenant isolation, time-series cost analytics, caching strategy, and an MCP-native integration layer for AI tooling.

</details>

<details>
<summary><b>🎗️ Breast Cancer Classification — PyTorch</b></summary>
<br/>

Feedforward neural network classifying tumors as malignant or benign on the Wisconsin Breast Cancer dataset.

| Stack | Scale | Performance | Security | Impact | Repository |
|---|---|---|---|---|---|
| PyTorch, Scikit-learn, FastAPI, Gradio | 569 samples, 30 features | 97.4% test accuracy, no significant overfitting | Standardized inference pipeline | Deployed demo on Hugging Face Spaces | [View Repo](https://github.com/shashwat-sahuu) |

Architecture: Linear-ReLU-Linear-Sigmoid, trained with Adam optimizer and BCE loss over 100 epochs, with StandardScaler preprocessing. Production version adds SHAP explainability, FastAPI inference, and Gunicorn multi-worker deployment.

</details>

<details>
<summary><b>🎯 Sonar Rock vs. Mine Detection — Classical ML</b></summary>
<br/>

Binary classifier distinguishing sonar signals reflected from rocks vs. metal mines.

| Stack | Scale | Performance | Security | Impact | Repository |
|---|---|---|---|---|---|
| Python, Scikit-learn, Pandas, NumPy | 208 samples, 60 frequency features | 76.2% test accuracy (83.4% train) | Stratified 90/10 split for class balance | Real-time single-instance prediction pipeline | [View Repo](https://github.com/shashwat-sahuu) |

Built to explore signal-based classification problems and establish a baseline classical-ML workflow before moving to deep learning approaches.

</details>

<details>
<summary><b>💊 SGLT2 / IgA Nephropathy — In Silico Molecular Docking</b></summary>

Computational drug discovery study docking candidate ligands against the SGLT2 target relevant to IgA Nephropathy.

| Stack | Scale | Performance | Security | Impact | Repository |
